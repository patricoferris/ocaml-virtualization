# virtualization

Bindings to Apple's virtualization framework

## Installation

### Using Opam

```bash
opam install inquire
```

### Using Esy

```bash
esy add @opam/inquire
```

## Usage

### In OCaml

```ocaml
let () = Virtualization.printf "World"
```
